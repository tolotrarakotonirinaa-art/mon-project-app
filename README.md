# 🚀 Mon Project App

## 📌 Description
Ce projet est une application développée avec React et déployée via Docker.

Elle fait partie d'une architecture avec :
- 🧪 Environnement de test (staging)
- 🚀 Environnement de production

---

## 🧪 Test (Staging)

- URL : http://localhost:8081
- Utilise Docker + Nginx
- Sert à tester les nouvelles fonctionnalités avant production

---

## 🚀 Production

- Environnement stable et sécurisé
- Utilisé par les utilisateurs finaux
- Déploiement contrôlé
