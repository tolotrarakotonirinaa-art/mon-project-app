# DevEnviron - Plateforme de Développement Environnemental

DevEnviron est une plateforme de développement collaborative complète qui permet aux équipes de gérer des projets, des tâches, des pipelines CI/CD, des dépôts Git, des environnements, et bien plus encore.

## 🚀 Fonctionnalités

- **Authentification** : Connexion et inscription avec différents rôles (Admin, Développeur, Client)
- **Tableau de bord** : Vue d'ensemble des projets, tâches et activités récentes
- **Gestion de projets** : Création, modification et suivi des projets
- **Gestion de tâches** : Kanban avec drag & drop, priorités, assignations
- **Pipeline CI/CD** : Visualisation et contrôle des étapes de développement
- **Dépôts Git** : Gestion des dépôts, visualisation et clonage
- **Environnements** : Gestion des environnements de développement, staging, production
- **Espace développeur** : Terminal intégré, éditeur de code, outils de test
- **Documentation** : Wiki intégré pour la documentation des projets
- **Communication** : Chat en temps réel entre les membres
- **Statistiques** : Graphiques et métriques sur les projets et tâches
- **Administration** : Gestion des utilisateurs et des permissions
- **Paramètres** : Configuration du profil, notifications, sécurité

## 🛠️ Technologies utilisées

- **React 18** - Bibliothèque UI
- **Context API** - Gestion d'état
- **CSS3** - Styles et animations
- **Font Awesome** - Icônes
- **Google Fonts (Poppins)** - Typographie
